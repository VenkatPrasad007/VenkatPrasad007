## 🚀 Featured Project — TaskFlow Pro

A production-grade full stack task management system built from scratch
following real-world engineering standards.

**Backend — Java 17 · Spring Boot 3 · PostgreSQL**
- RESTful API with clean layered architecture (Controller → Service → Repository)
- DTOs for request/response separation — no raw entities exposed
- Global exception handling with standardized error responses
- JPA/Hibernate with auto-managed timestamps
- Swagger/OpenAPI interactive documentation
- Environment-based configuration — zero hardcoded credentials

**Frontend — React 18 · TailwindCSS · Axios**
- Dashboard with priority and status color-coded badges
- Complete CRUD — Create, Edit, Delete tasks directly from UI
- Filter tasks by status with live task count
- Toast notifications, loading states, empty states
- Custom React hooks and centralized Axios API client

**AI Integration (In Progress)**
- Smart Task Assistant powered by Claude AI
- Generates actionable tasks from plain English goals
- Full stack AI integration — React → Spring Boot → Claude API
